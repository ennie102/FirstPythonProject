# Beginner Python Projects

This repository contains my very first Python programming exercises completed during the early stages of my **AI Software Development Learnership** at Synergy Transformation Solutions.

### Project Overview

This notebook showcases the foundational Python skills I learned as I began my journey into programming and Artificial Intelligence. It includes several simple, interactive programs that demonstrate core programming concepts such as user input, data type conversion, conditional statements, loops, and basic arithmetic operations.

These beginner exercises helped me build confidence in writing clean Python code and understanding logical program flow — skills that now serve as the foundation for more advanced Machine Learning and AI projects.

### What I Built

The notebook includes the following exercises:

- **Personalized Greeting Program** – Asks for the user’s name and prints a friendly welcome message.
- **Basic Addition Calculator** – Takes two numbers from the user and returns their sum.
- **Personal Information Collector** – Collects first name, last name, and age, then displays a formatted summary.
- **Even or Odd Number Checker** – Determines whether an entered number is even or odd using conditional logic (`if-else`).
- **Countdown Timer** – Accepts a number and counts down to 1, ending with “Blast off!” using a `for` loop.
- Additional simple input and output practice scripts.
